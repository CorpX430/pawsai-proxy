📊 API Endpoint

Use this proxy to fetch football data with CORS support:

/api/football?path=/v4/competitions/PL/matches&status=SCHEDULED

🔧 Example Usage

fetch('/api/football?path=/v4/competitions/PL/matches')
  .then(res => res.json())
  .then(data => console.log(data));

🏆 Available Leagues

PL (Premier League), SA (Serie A), PD (La Liga), BL1 (Bundesliga), FL1 (Ligue 1), CL (Champions League)
